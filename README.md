# CAMSim - Flight Simulator that implements JSBSim XML format

## Features

Parses JSBSim XML format specification: https://jsbsim.sourceforge.net/JSBSim_Config_File_Format_v2.0.pdf
Full support for JSBSim XML format

## Usage

```bash
go run main.go
```
